### 步骤5：选择选项

**操作**：点击选项按钮，例如"A. 自然风光"

**界面变化**：
- 选项被选中，显示蓝色高亮
- 节点状态变为"selected"
- 显示"已选择: 自然风光"
- 如果是前两个节点，自动生成下一个思考节点
- 如果是第三个节点，自动转入执行蓝图

**代码位置**：`src/store/useBlueprintStore.ts` 第167-188行

```tsx
// useBlueprintStore.ts - selectThinkingOption
selectThinkingOption: (nodeId, optionId) => {
  set((state) => {
    const updatedNodes = state.thinkingNodes.map((node) => {
      if (node.id === nodeId) {
        return { ...node, status: 'selected', selectedOption: optionId };
      }
      return node;
    });
    
    // 如果不是最后节点，生成下一节点
    if (state.currentThinkingIndex < 2) {
      const nextNode = generateThinkingNode(state.currentThinkingIndex + 1);
      return { 
        thinkingNodes: [...updatedNodes, nextNode],
        currentThinkingIndex: state.currentThinkingIndex + 1 
      };
    }
    
    // 第三个节点完成后转入执行
    return { thinkingNodes: updatedNodes };
  });
}
```
