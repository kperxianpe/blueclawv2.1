# Blueclaw 前端手动测试指南

## 测试环境确认
- ✅ 前端: http://localhost:5174 (运行中)
- ✅ 后端: ws://localhost:8006 (运行中)

---

## 📋 测试题目与截图要求

### 测试 1: 页面加载与 WebSocket 连接

**操作步骤：**
1. 打开 Chrome 浏览器
2. 访问 http://localhost:5174
3. 按 F12 打开开发者工具 → Console 标签
4. 等待 3 秒

**截图要求：**
- [ ] 完整页面截图（显示 Blueclaw 标题和输入框）
- [ ] 控制台截图（显示 WebSocket 连接状态）

**预期检查结果：**
- 控制台显示 `[WebSocket] Connected` 或 `[WebSocket] Connecting`
- 没有红色错误信息

---

### 测试 2: 输入功能

**操作步骤：**
1. 在输入框输入：`帮我规划杭州2日游`
2. 点击"开始"按钮

**截图要求：**
- [ ] 输入后的输入框截图
- [ ] 点击后思考界面截图（显示思考节点）

**预期检查结果：**
- 左侧显示"思考蓝图"面板
- 出现第一个思考节点（问题：想规划什么样的旅行？）

---

### 测试 3: 思考节点选择

**操作步骤：**
1. 点击第一个思考节点（展开显示选项）
2. 选择"自然风光"选项
3. 等待 2 秒

**截图要求：**
- [ ] 展开的思考节点截图（显示3个选项）
- [ ] 选择后的节点状态截图
- [ ] 显示第二个思考节点的截图

**预期检查结果：**
- 第一个节点显示"已选择"状态
- 下方出现第二个思考节点

---

### 测试 4: 思考收敛与执行确认

**操作步骤：**
1. 依次完成3个思考节点的选择
2. 观察最后一个节点选择后的状态

**截图要求：**
- [ ] 3个节点都选择完成的截图
- [ ] 绿色"思考完成，开始执行"按钮截图
- [ ] 点击按钮后的执行蓝图截图

**预期检查结果：**
- 显示绿色确认按钮
- 执行蓝图显示多个执行节点（查询天气、搜索景点等）

---

### 测试 5: Adapter 系统

**操作步骤：**
1. 进入执行蓝图界面
2. 观察左侧中间的 Adapter 列表
3. 尝试拖拽左侧黄色方块到右侧执行节点

**截图要求：**
- [ ] 左侧 Adapter 列表面板截图（显示黄色方块）
- [ ] 绑定 Adapter 后的执行节点截图

**预期检查结果：**
- 左侧显示 Web Search、Image Gen 等 Adapter
- 节点右上角显示绑定的 Adapter 图标

## 🔧 故障排查

### 问题1: 页面空白
**解决：** 刷新页面，检查控制台报错

### 问题2: WebSocket 连接失败
**解决：** 
- 检查后端是否运行：`netstat -ano | findstr 8006`
- 刷新页面重试

### 问题3: 点击无反应
**解决：** 
- 检查是否在正确的阶段操作
- 查看控制台是否有错误
