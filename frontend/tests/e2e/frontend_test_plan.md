# Blueclaw 前端测试计划

## 测试环境
- 前端: http://localhost:5174 (5173被占用)
- 后端: ws://localhost:8006 (保持运行)

## 测试题目 1: 页面加载与 WebSocket 连接

### 操作步骤
1. 打开浏览器访问 http://localhost:5174
2. 按 F12 打开控制台
3. 查看 Console 标签页

### 预期结果
- [ ] 页面正常加载，显示 Blueclaw 标题和输入框
- [ ] 控制台显示 `[WebSocket] Connected` 或连接状态
- [ ] 没有红色报错信息

### 截图检查点
- 整个页面截图
- 控制台截图
